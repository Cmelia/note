## 1.1. git  status 中文显示乱码

解决 :  git config --global core.quotepath false

## 1.2. 新建文件

在本地版本库新建文件后，要执行`git add wenjian.py`,`git commit -m "shuoming"`和`git push origin master`
## 1.3. push 与 pull

### 1.3.1. push

在本地进行修改文件后，点击右侧“源代码管理”，保存修改的文件；再点击右上角的三个点下拉菜单中的“推送”即可。
